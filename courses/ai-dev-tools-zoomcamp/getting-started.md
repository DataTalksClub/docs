# Getting Started

For the cross-course onboarding, read [Joining a Cohort]({{ '/courses/zoomcamp-logistics/joining/' | relative_url }}) first. It covers registration, account setup, calendar, and newsletter updates.

This page covers the AI Dev Tools Zoomcamp specifics.

## Star the GitHub repository

[github.com/DataTalksClub/ai-dev-tools-zoomcamp](https://github.com/DataTalksClub/ai-dev-tools-zoomcamp)

All course materials are here. Each module has its own folder (for example `01-overview`, `03-mcp`), and cohort-specific homework and deadlines are under the `cohorts` folder.

## Join the Slack channel

After joining the [DataTalks.Club Slack workspace](https://datatalks.club/slack), find the `#course-ai-dev-tools-zoomcamp` channel. This is where questions, announcements, and discussion happen. Before asking, read [Asking Questions]({{ '/courses/zoomcamp-logistics/asking-questions/' | relative_url }}).

## Follow the announcements

- Join the [course Telegram channel](https://t.me/aidevtoolszoomcamp).
- Subscribe to the [DataTalks.Club YouTube channel](https://www.youtube.com/@DataTalksClub) and check the [course playlist](https://www.youtube.com/playlist?list=PL3MmuxUbc_hLuyafXPyhTdbF4s_uNhc43).

## How to work through the course

1. Start with the Q&A / launch session and Module 1 (Introduction / Vibe Coding).
2. Read the README in each module folder for an overview and video links.
3. Watch the lectures (all pre-recorded) and do the homework.
4. Finish with the final project, then review three peers' projects.

See [Environment Setup]({{ '/courses/ai-dev-tools-zoomcamp/environment-setup/' | relative_url }}) for the tools you'll use.
