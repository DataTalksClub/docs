# Slack

Use Slack for anything that needs a reply:

- Course questions.
- Homework or project clarifications.
- Tool errors and setup problems.
- Office-hours follow-ups.
- Sharing progress and helping peers.

Join the [DataTalks.Club Slack workspace](https://datatalks.club/slack.html), then join the course-specific channel.

Slack gets busy during live cohorts, so use threads. For how to ask well, see [Asking Questions]({{ '/courses/zoomcamp-logistics/asking-questions/' | relative_url }}). For Slack norms that apply across DataTalks.Club, see the general [Slack guide]({{ '/general/slack/' | relative_url }}).
